int mcp_thermocouple_enable ( i2c_thermocouple *  pdevice)

mcp_thermocouple_enable() takes a reference to an inactive i2c_thermocouple structure and enabled it, returning 0 on success. The function sleeps for half a second before returning to account for device sampling time.
