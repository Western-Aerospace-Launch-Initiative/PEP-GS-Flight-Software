#define HALF_SECOND   500000

HALF_SECOND is a macro that references an int valuew 500000. This is used in the stdlib usleep() function to sleep for 500000 microseconds or half a second. This allows for the thermocouple to wake from shutdown mode before giving access back to the user to prevent trying to read the device termperature before the temperature register can update.
